Development environment

Our development environments are highly portable. Each project/website lives within its own virtual machine, which means there is little to install on your development machine.

Applications

You'll need the following applications.

  • Google Chrome
  • Slack
  • Atom
  • Git
  • Vagrant
  • Sequel Pro
  • cord
  • VMWare Fusion
  • Kaleidoscope
  • Transmit

Accounts

You'll need the following accounts.

  • Google account
  • Apple account
  • GitHub account
  • Dropbox account

Virtual machines

Each and every project runs within its own virtual machine. Although there are hits on memory and file space the major upside is reproducability - not to the mention the fact you can delete and recreate entire development environments within 15 minutes.

The following shows the typical workflow:

  • [host] Write some code
  • [vm] Run tests
  • [host] View in browser
  • [host] Commit changes into Git

All development resources such as databases, web servers, application servers (Node.js) are run inside the VM.
